The Expense Tracker is a simple and user-friendly web application that helps users track their income and expenses. It calculates the total balance automatically and displays transaction history.
This project is built using HTML, CSS, and JavaScript.

Add income and expense transactions
- Automatic balance calculation
- Separate income and expense totals
- Transaction history display
- Delete transactions
- Real-time updates without page refresh

Through this project, I improved my understanding of:
- JavaScript fundamentals
- Working with the DOM
- Handling user input
- Updating UI dynamically
- Building small real-world applications

Key Concepts Used
- Event Listeners
- Array Methods (map, filter, reduce)
- DOM Manipulation
- Form Handling
- Dynamic UI Updates
