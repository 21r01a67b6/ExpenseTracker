# ExpenseTracker

Project Overview
The Expense Tracker is a simple and user-friendly web application that helps users track their income and expenses. It calculates the total balance automatically and displays transaction history.
This project is built using HTML, CSS, and JavaScript.


How It Works
1. User enters description and amount.
2. If the amount is positive → Income.
3. If the amount is negative → Expense.
4. Balance is calculated as:

   Balance = Total Income - Total Expense

5. Transactions can be removed using the delete button.
